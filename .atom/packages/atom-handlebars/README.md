# Handlebars support in Atom

Adds syntax highlighting and snippets to Handlebars files in Atom.

## Install

```bash
$ apm install atom-handlebars
```

Or Settings ➔ Packages ➔ Search for `atom-handlebars`

## Snippets

- `each` ⇥
- `list` ⇥
- `if` ⇥
- `ifelse` ⇥
- `unless` ⇥
- `unelse` ⇥
- `else` ⇥
- `with` ⇥
- `log` ⇥
- `dbg` ⇥
- `escape` ⇥
